## Скачать и установить

- **Python**: [Скачать Python 3.13.13](https://www.python.org/ftp/python/3.13.13/python-3.13.13-amd64.exe)  
  _Важно: поставить галочку "Add Python to PATH"_
- **Git**: [Скачать Git](https://git-scm.com/install/)
- **VS Code**: [Скачать VS Code](https://code.visualstudio.com/download)  
  _Важно: поставить 5 галочек_
- **GitHub Desktop**: [Скачать GitHub Desktop](https://desktop.github.com/download/)
