# Курс по Python без фреймворков

Этот репозиторий — структурированный курс по Python без фреймворков: от синтаксиса, модели объектов и работы с памятью до системного программирования, сети, асинхронности, качества кода, архитектуры и практик разработчика.

Курс организован как дерево модулей и тем. Верхний уровень — модули `0100`, `0200`, `0300` и далее. Внутри модулей темы разбиты на отдельные `.py`-файлы с последовательной нумерацией. Для крупных прикладных блоков используются вложенные папки по библиотекам или подсистемам.

## Как читать этот репозиторий

1. Идите по модулям сверху вниз: от `0100` к следующим блокам.
2. Внутри модуля читайте темы по числовой нумерации.
3. Используйте файлы как каркас курса, конспекты и точки расширения под примеры и задания.
4. Для библиотечных блоков смотрите вложенные папки по конкретным модулям стандартной библиотеки и экосистемы.

## Принципы курса

- Основа курса — Python и стандартная библиотека.
- Фреймворки не используются как опорная часть программы.
- Темы организованы от языка и модели исполнения к инженерным практикам.
- Нумерация файлов и папок отражает порядок изучения.
